# 🌍 WanderLust – Travel Listing Web App

## 🚀 Overview

WanderLust is a full-stack web application inspired by platforms like Airbnb.
Users can explore travel listings, create their own listings, and share experiences with reviews.

---

## ✨ Features

* 🏡 Create, edit, and delete travel listings
* 🔐 User authentication & authorization
* 📝 Add reviews and ratings
* 📸 Image uploads with cloud storage
* 📍 Location-based listings
* ⚡ Clean MVC architecture

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, EJS
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **Authentication:** Passport.js
* **Cloud Storage:** Cloudinary
* **Others:** Mongoose, Joi Validation

## ⚙️ Installation & Setup

1. Clone the repository

```
git clone https://github.com/ashwathjelloji2468-gif/wanderlust.git
```

2. Navigate to project folder

```
cd wanderlust
```

3. Install dependencies

```
npm install
```

4. Create a `.env` file and add:

```
DB_URL=your_mongodb_connection
SECRET=your_secret_key
CLOUDINARY_CLOUD_NAME=your_cloud_name
CLOUDINARY_KEY=your_key
CLOUDINARY_SECRET=your_secret
```

5. Run the application

```
node app.js
```

6. Open in browser:

```
http://localhost:3000
```
